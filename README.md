## Descrição
Aplicação web de **conversão de moedas em tempo real**, desenvolvida com foco em **simplicidade**, **usabilidade** e **escalabilidade**.
O projeto permite converter valores do **Real Brasileiro (BRL)** para diversas moedas internacionais, utilizando **dados atualizados via API**, além de apresentar informações visuais como nome da moeda e bandeira do país selecionado.



## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Conversão de valores do Real Brasileiro (BRL) para múltiplas moedas internacionais

- Utilização de API de câmbio para obter valores atualizados em tempo real

- Atualização dinâmica do nome da moeda e da bandeira do país conforme a seleção

- Formatação automática dos valores de acordo com o padrão regional de cada moeda

- Interface responsiva e intuitiva

- Conversão automática ao alterar a moeda selecionada
